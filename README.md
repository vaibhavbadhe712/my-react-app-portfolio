# Vaibhav Badhe Portfolio

A modern, responsive, and animated portfolio website for a Senior Flutter Developer.

## Tech Stack
- React 18
- Vite
- Framer Motion (Animations)
- Lucide React (Icons)
- CSS3 (Custom Glassmorphism Design)

## Features
- Fully Responsive (Mobile First)
- Dark Theme with Gradient Accents
- Smooth Scroll Animations
- Animated Project Cards
- Contact Form UI

## How to Run

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Start the development server:
    ```bash
    npm run dev
    ```

3.  Build for production:
    ```bash
    npm run build
    ```

## Customization
- Update `src/components/` files to change content.
- Update `src/index.css` for global theme variable adjustments.
